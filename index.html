Notes: Memory location $fe contains a new random byte on every instruction. Memory location $ff contains the ascii code of the last key pressed. Memory locations $200 to $5ff map to the screen pixels. Different values will draw different colour pixels. The colours are: $0: Black $1: Red $2: Green $3: Yellow $4: Blue $5: Purple $6: Dark Cyan $7: Grey $00 - $07: Regular colours $08 - $0f: Saturated colours $10 - $17: Brighter colours $18 - $1f: Brighter and saturated colours $20 - $80: ASCII characters $81 - $bf: 6-bit colour depth 64 colours $c0 - $ff: greyscale Use keyword "define" to assign names to values. Use keyword "dcb" to define constant literal bytes. Hex notation is $xx and binary notation is bxxxxxxxx.
